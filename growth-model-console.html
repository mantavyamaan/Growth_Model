Guardrails (anti-gaming)

Rule A — Repeatable low-value actions are capped per period (tasks ≤ 20, peer-help ≤ 10, CRM batches ≤ 8, follow-ups ≤ 15). Capped fields glow amber.
Rule B — Every scored action should have evidence (ticket, approval, CRM record, audit log, artifact).
Rule C — No double counting; one action scores in one category.
Rule D — Major negatives only; small mistakes aren't over-punished.
