# Growth Model

A mathematical framework for measuring employee growth through contribution-based performance metrics. You can acess the model here: 
[Growth Model](https://mantavyamaan.github.io/Growth_Model/growth-model-console.html)

## Overview

The Growth Model is a role-aware performance evaluation framework designed to measure employee growth using measurable workplace contributions rather than subjective ratings. It quantifies impact across multiple performance pillars while maintaining fairness, transparency, and scalability.

Unlike traditional appraisal systems that rely heavily on manager feedback, this model rewards measurable achievements, meaningful contributions, and business impact through an additive scoring approach.

## Features

- 📈 Contribution-based employee growth evaluation
- 🎯 Role-specific scoring for different job functions
- ⚖️ Transparent and explainable scoring methodology
- 🚀 Rewards measurable business impact and ownership
- 🛡️ Built-in anti-gaming guardrails
- ➕ Bonus framework for exceptional achievements
- 📊 Scalable and customizable scoring system

## Performance Pillars

The model evaluates growth across multiple universal dimensions, including:

- Business Impact
- Execution & Delivery
- Collaboration
- Learning & Development
- Innovation & Ownership
- Leadership & Influence

Each pillar contains measurable metrics tailored to specific organizational roles.
